## About The Project

This is a simple application with the concept of a trending food sale.

### Built With

- [Node Js v14.19.1 or Latest](https://nodejs.org/)
- [Javascript](https://www.w3schools.com/js/DEFAULT.asp)
- [React Js](https://reactjs.org/)
- [Next Js](https://nextjs.org/)
- [Redux](https://redux.js.org/)
- [Chakra UI](https://chakra-ui.com/)
- [Tailwind CSS](https://tailwindcss.com/)

### Installation

1. Install Node Js

    <div align="center">
     <img width="500" src="./public/assets/installer/node.PNG">   
    </div>

   ```
   Install for windows latest version
   ```

   <div align="center">
     <img width="500" src="./public/assets/installer/previous-rilis.PNG">   
   </div>

   <div align="center">
     <img width="500" src="./public/assets/installer/previous-list.PNG">   
   </div>

   ```
   Install for windows previous version
   ```

   <div align="center">
     <img width="500" src="./public/assets/installer/install-node.PNG">   
   </div>

   ```
   Install Node Js
   ```

2. Install yarn

   <div align="center">
     <img width="500" src="./public/assets/installer/yarn.PNG">   
   </div>

   ```
   install global yarn with "npm install --global yarn" and check version "yarn --version"
   ```

3. Clone the repo

   <div align="center">
     <img width="500" src="./public/assets/installer/clone.PNG">   
   </div>

   ```
   git clone https://github.com/BudiSetyo/warung4.0.git
   ```

4. Open the folder

5. Install dependencies

   ```
   yarn install
   ```

6. Run the App

   ```
    yarn dev
   ```
